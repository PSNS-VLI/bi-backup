# GxTitle标题

一个神奇的标题组件

## Examples

### 简单用法

```vue
<template>
  <GxTitle> 这是标题 </GxTitle>
</template>
```

## API

### <a name="Attributes">Attributes</a>

| 属性名 | 说明       | 类型   | 可选值            | 默认值  |
| ------ | ---------- | ------ | ----------------- | ------- |
| icon   | 标题前图标 | String | [GxIcon](#GxIcon) | icon-xz |

### <a name="Slots">Slots</a>

| 插槽名  | 说明     |
| ------- | -------- |
| default | 标题文字 |

### <a name="GxIcon">GxIcon</a>

> GxIcon源于iconfont,可选值请查看[icon.scss](../icon.scss),注意icon-这种前缀已经不需要传递了。
